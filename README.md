### Troubleshoot

#### Linux:

1. Mason workaround:

Add this in `mason-workaround.lua` file in plugins dir if linux mason version throws an error:
```lua
return {
  { "mason-org/mason.nvim", version = "^1.0.0" },
  { "mason-org/mason-lspconfig.nvim", version = "^1.0.0" },
}
```
